Tags de Título (Heading)

<h1> a <h6>

Título H1

Título H2

Título H3

Título H4

Título H5
Título H6
Tags usadas para definir títulos e subtítulos, onde <h1> é o mais importante e <h6> o menos importante.

Formatação de Texto

<p>

Este é um parágrafo de texto. Ele é usado para agrupar conteúdos textuais.

Define um parágrafo de texto.
<mark>

Este texto contém uma parte destacada com a tag mark.

Usado para destacar parte do texto.
<small>

Texto normal e texto pequeno.

Define texto menor, geralmente para notas ou direitos autorais.
<i>

Texto em itálico para ênfase ou termos técnicos.

Usado para texto em itálico, geralmente para ênfase ou termos técnicos.
<u>

Texto sublinhado para indicar nomes próprios ou erros de ortografia.

Usado para sublinhar texto.
<strong>

Texto com importância forte.

Usado para dar forte importância ao texto (normalmente exibido em negrito).
<font>

Texto com formatação específica usando a tag font.

Tag obsoleta que era usada para especificar fonte, cor e tamanho do texto. Hoje em dia usamos CSS para isso.
<del>

Preço: R$ 100,00 R$ 80,00

Representa texto que foi removido do documento.
<abbr>

A WWW foi inventada por Tim Berners-Lee.

Define uma abreviação ou acrônimo. O atributo title contém a expansão do termo.

Listas

<ul> e <li>

Lista não ordenada:

  • Item 1
  • Item 2
  • Item 3
<ul> cria uma lista não ordenada (com marcadores) e <li> define cada item da lista.
<ol> e <li>

Lista ordenada:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
<ol> cria uma lista ordenada (numérica ou alfabética) e <li> define cada item da lista.

Links

<a>

Visite o Google para fazer pesquisas.

Define um hyperlink. O atributo href especifica o URL de destino.

Outras Tags

<hr>

Texto acima da linha

Texto abaixo da linha

Cria uma linha horizontal para separar conteúdos.
<sub>

Fórmula da água: H2O

Define texto subscrito (ligeiramente abaixo da linha normal).
<sup>

Teorema de Pitágoras: a2 + b2 = c2

Define texto sobrescrito (ligeiramente acima da linha normal).
<blockquote>

Citação famosa:

"A vida é o que acontece enquanto você está ocupado fazendo outros planos."
Define uma citação longa de outra fonte. O atributo cite especifica a fonte da citação.
